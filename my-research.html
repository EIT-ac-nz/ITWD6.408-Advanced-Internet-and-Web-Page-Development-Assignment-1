Copyright, Creative Common (CC) Licences, Fair Use



Originality: In order to get a copyright, the work must be the original work from the original author. This means the work shouldn’t be a copy of other works.

Creativity: A work will only need a small quantity of creativity in order to apply for copyright.

Fixation: In order for works to have copyrights, they do not last for a very long time unless there is a record of said work in which the copyright can be applied as it is considered a fixation.

Types of CC: Attribution, Share Alike, NonComercial and No Derivatives

WEB PERFORMANCE AND MAINTENANCE




Ways to improve page loading speed:

Reduce the amount of redirects your website uses, this makes loading easier as it will not have to jump as far.
Organize and minimize the amount of code used, having less code that will direct you instantly without loading in many other things will increase speed by a bit or a lot depending on how much code there was/is.
Choosing the right hosting provider will affect the speed, this mostly depends on the provider so choosing the right one matters.
Reduce the amount of plugins used, plugins tend to be unnecessary and slow down the website so deactivating the ones that aren't needed help speed things up.
Optimizing images so they dont take up as much space to load, having less pixels to load can add to the websites loading speed but you dont need to lower it all the way.
Enabling the browser chaching, this lets the website store information making it easier to revisit once its been loaded once.


Important ways to manage your website:

Reduce the amount of redirects your website uses, this makes loading easier as it will not have to jump as far.
Keeping updated backups of the website, keeping multiple backups on the website allows the website to be restored to a previous point if something were to go wrong and just in case, the further back the safer it is even though some things may be missing.
Monitoring your websites uptime and downtime, if the website goes down it will cost you a lot expecially in a business that runs through the website. Keeping track of when the site goes down can allow you to quickly act to get it running again.
Constant website testing, testing the website is important as if it can be broken it will have a big negative impact. You can run tests to see if the website can be broken or if any loopholes are present and this increases the security of the website as knowing what could happen and fixing the problems will keep your website safe.
Managing content loading times, ensuring that the websites content can be loaded in a timely matter when the website has many visitors will keep the website running at a good pace and not affect the many visitors browsing in a heavy manner.
Checking feedback, keeping updated on the new comments and activities of the visitors will help in knowing what they are thinking so you can adjust and make changes that could satify them.

WEB SECURITY




5 types of cyber attacks and how to deal with them:

Malware attacks: These are malicious software attacks that are designed to exploit and embed itself into the system it has been installed on. These can be avoided by using antivirus softwares that will scan the files before you open them reducing the chance of the file being malicious.
Password attacks: These attacks are designed to attempt to crack and gain access to the device and any other authentication medthods by running a command that will spam all combinations untill it suceeds. Picking a long and complication password can help reduce the time it takes for it to be cracked but only decreases the chances of it succeding, some websites have multi-authentication menthods which can improve the security of the account drastically.
DDOS attacks: These attacks use other compromised systems to target a specific system in order to flood it with traffic that will attempt to crash or shut down the target. These can be avoided using firewalls and network security that will scan incoming traffic and block it.
Phishing attacks: These attacks use fraudulent information to spread malicious links or information through spam that can trick unsuspecting victims. These can be avoided using anti phishing and spam softwares that will protect you from malicious threats that come through the messages/emails on your computer.
DNS Spoofing attacks: These attacks use the DNSs of servers to overwrite the IP and direct victims to malicious websites through the targeted website. These can be avoided using DNS security extentions, applying patches and performing DNS traffic filtering that will help prevent your servers DNS being redirected.
